🥘 ÍzKalauz Pro – WPF Desktop Kliens
Az ÍzKalauz egy prémium menütervező és receptkezelő alkalmazás, amely segít a heti étrend összeállításában és a receptek rendszerezésében.
Ez a repozitórium a projekt WPF (C#) alapú asztali kliensét tartalmazza.

🚀 Funkciók
Receptböngésző: Teljes receptlista lekérése a Backend API-ból.

Kategória szűrés & Keresés: Gyors keresés név alapján és szűrés ételtípusok (leves, főétel, desszert stb.) szerint.

Részletes receptnézet: Megújult, vizuális felület a hozzávalók és az elkészítési útmutató megtekintéséhez.

Heti menütervező: Receptek hozzárendelése a hét napjaihoz, intelligens duplikáció elleni védelemmel.

Exportálás: A kész heti étrend mentése .txt formátumba az asztalra.

🛠️ Technikai részletek
Keretrendszer: .NET / WPF (Windows Presentation Foundation)

Adatkezelés: Aszinkron REST API kommunikáció (HttpClient, Newtonsoft.Json).

UI Design: Modern, kártya-alapú elrendezés árnyékokkal és színátmenetes háttérrel.

Verziókezelés: Git & GitHub.
